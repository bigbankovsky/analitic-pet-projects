# analitic-pet-projects
Это мои pet-проекты, которые я делал пока учился на аналитика. Проекты сделаны с целью показать базовые навыки в анализе данных. В каждом ноутбуке вначале идет код, который скачивает необходимые датасеты прям из ноутбука. Для этого вам понадобится API-ключ из личного кабинета kaggle. Для удобства эти же датасеты я загрузил в репозиторий, так что можете пропускать эти участки кода. 
В данных проектах я затронул следующие темы: А/B-тесты, статистика, когортный анализ, дашборды, RFM-анализ, графики.
